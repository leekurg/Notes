### About ###
Swift/UIKit app for creating notes

#### Features: ####
  - Create and store notes in local database
  - Edit and delete notes
  - Choose note color and category 

#### Technology stack: ####
  - Swift/UIKit
  - Architecture MVC
  - Realm Swift database
  - Dependencies with CocoaPods

### Installation ####
```sh
   cd .../Notes
   pod install 
```
